# Portfolio — Poloko Nkolanyane

Terminal-themed personal portfolio built with **React + Vite**. Features a boot sequence, scroll-reveal animations, and sections for about, skills, experience, projects, and contact.

## Tech Stack

- **React 18** — UI framework
- **Vite 6** — Build tool
- Pure CSS — no UI library

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Sections

- `About` — summary and background
- `Skills` — categorized tech stack
- `Experience` — work history
- `Projects` — featured projects
- `Contact` — links and contact info

## Data

All content is driven by `src/data/portfolio.json`. Edit that file to update your profile without touching components.
